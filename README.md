# 📚 Kado Ceria

**Kado Ceria** adalah aplikasi mobile interaktif yang dirancang khusus untuk mendukung pembelajaran bahasa Indonesia dan bahasa Sunda bagi:

- 👶 Anak usia dini  
- 🌏 Pembelajar BIPA (Bahasa Indonesia bagi Penutur Asing)  
- ♿ Penyandang disabilitas (khususnya tunanetra dan tunarungu)  

Aplikasi ini merupakan hasil alih wahana dari buku digital cerita anak dwibahasa terbitan **BBPJB** menjadi **buku audio interaktif**.

---

## ✨ Fitur Utama

- 📖 **Daftar Cerita Anak**  
  Menampilkan berbagai pilihan cerita anak dalam dua bahasa (Indonesia & Sunda).

- 📘 **Flipbook Interaktif**  
  Mengubah PDF menjadi tampilan flipbook dengan animasi halaman yang realistis.

- 🔊 **Narasi Audio Otomatis**  
  Putar narasi dari awal hingga akhir, disesuaikan dengan bahasa yang dipilih pengguna.

- 👆 **Interaksi Objek**  
  Ketuk objek tertentu untuk mendengar bunyi/kata terkait objek tersebut.

- 🔁 **Navigasi Bebas Halaman**  
  Pengguna dapat berpindah ke halaman lain meskipun audio masih berjalan.

- 🌐 **Dukungan Multibahasa**  
  Bahasa Sunda dan Bahasa Indonesia sebagai pilihan bahasa narasi & UI.

---

## 🧑‍💻 Teknologi yang Digunakan

- **Flutter** — Framework utama untuk pengembangan aplikasi mobile (Android & iOS).
- **Arsitektur MVVM** — Memisahkan logika tampilan dan data untuk pemeliharaan yang lebih baik.

---

## 🎯 Tujuan Aplikasi

Sebelum kehadiran **Kado Ceria**, pembelajaran dilakukan dengan:
- Buku cetak + audio terpisah untuk anak usia dini dan pembelajar BIPA.
- Buku braille untuk penyandang disabilitas.

Dengan aplikasi ini, diharapkan proses pembelajaran menjadi:
- **Lebih terintegrasi dan menyenangkan**
- **Aksesibel untuk berbagai kalangan**
- **Mendukung pelestarian bahasa daerah**

---

## 👥 Karakteristik Pengguna

1. **Anak Usia Dini**  
   Antarmuka yang menarik, interaktif, dan mudah digunakan untuk usia pra-sekolah.

2. **Pembelajar BIPA**  
   Cocok bagi penutur asing yang ingin mempelajari bahasa Indonesia dan bahasa daerah.

3. **Penyandang Disabilitas**  
   - **Tunanetra**: Mengandalkan narasi audio interaktif.  
   - **Tunarungu**: Fokus pada visualisasi cerita dan animasi interaktif.

---

## 🚀 Cara Menjalankan Aplikasi

> **Catatan:** Petunjuk ini disesuaikan untuk proyek Flutter.

```bash
# Clone repositori
git clone https://github.com/namarepo/kado-ceria.git
cd kado-ceria

# Install dependency
flutter pub get

# Jalankan aplikasi
flutter run
```
